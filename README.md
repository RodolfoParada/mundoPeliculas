### Mundo Películas: Un catálogo dinámico de anime y cine desarrollado como una Single Page Application (SPA) para gestionar contenido multimedia.

#### Objetivo: Proporcionar una interfaz fluida para explorar series, películas y estrenos con navegación por letras y paginación personalizada.

### Tecnologías Utilizadas

#### HTML5: Estructura semántica para la organización del contenido.
#### CSS3: Diseño responsivo utilizando variables personalizadas (:root), Flexbox y animaciones de aparición.
### JavaScript (Vanilla): Lógica para la paginación dinámica (6 ítems por página), filtrado por letra inicial y persistencia de datos mediante localStorage.

### Características Principales
#### Paginación Dinámica: Sistema que agrupa las películas de 6 en 6 para optimizar la carga visual.
#### Navegación Alfabética: Menú interactivo de la A a la Z que permite filtrar contenidos instantáneamente sin recargar la página.
#### Diseño Adaptable (Responsive): Optimizado para móviles, tablets y ordenadores mediante Media Queries específicas.
#### Persistencia con LocalStorage: El sitio recuerda la última página visitada por el usuario aunque cierre el navegador.
